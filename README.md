# Invoice Call Manager UI

Modern React + Tailwind CSS frontend for invoice management.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Upload CSV/XLSX/XLS files
- AI-powered parsing
- Real-time updates (3-second silent refresh)
- View call notes in modal
- Batch calling with one click
- Clean, modern UI with Tailwind CSS

## Environment

Create `.env`:
```
VITE_API_URL=http://localhost:8000
```

## Stack

- React 19 + TypeScript
- Tailwind CSS
- Axios
- React Icons
- Vite
